# Nuha Ahmed — Portfolio

A warm, professional single-page portfolio built with **Next.js 14**, **TypeScript**, and **Tailwind CSS**.

## Color Palette

| Token | Hex | Usage |
|---|---|---|
| Rose Blush | `#E8C4B8` | Accents, borders |
| Rose Muted | `#D4A096` | Bullets, tags |
| Rose Deep | `#B8756A` | Headings accent, CTAs |
| Cream | `#FAF3EE` | Background |
| Charcoal | `#2C3830` | Primary text |
| Charcoal Mid | `#3D5045` | Body text |

## Fonts

- **Cormorant Garamond** (Display / headings) — elegant, editorial
- **DM Sans** (Body) — clean, readable
- **DM Mono** (Labels, tags, code-like text)

## Sections

1. **Hero** — Name, title, tagline, CTAs, certifications
2. **Experience** — Timeline with all four roles
3. **Skills** — Three grouped skill cards + education strip
4. **Contact** — Links (email, phone, LinkedIn, GitHub)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Deploy

Easiest option: [Vercel](https://vercel.com) — push to GitHub and connect via `vercel.com/new`.

```bash
npm i -g vercel
vercel
```
